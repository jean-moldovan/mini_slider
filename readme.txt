Mini slider 1.0 - free jQuery image/content slider plugin (Deprecated)

Requires jQuery library - http://jquery.com

Features:
 - Free download
 - Lightweight
 - Browser support: Firefox, Chrome, Safari, iOS, Android, IE7+, Opera
 - HTML5 & CSS3 clean and valid markup
 - Slides can contain images or any HTML content
 - Horizontal slideshow
 - Autoplay mode
 - Prev/Next controls
 - Bullet navigation
 - Customizable size, speed, delay
